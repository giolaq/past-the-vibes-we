# TV Porting Plan

Start from the analysis. Describe the TV screen layout and the exact remote control flow: initial focus, directional movement and its boundaries, Select to open details, and Back to restore the originating item. Keep unsupported items as explicit gaps instead of inventing platform APIs. The plan is a document, not code.
