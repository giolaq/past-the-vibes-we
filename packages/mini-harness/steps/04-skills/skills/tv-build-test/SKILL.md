# TV Build and Test

Produce a build preview artifact, then test the ported flow the way a viewer would with a remote. Record a screenshot as evidence and the remote control checks in order: launch focus, directional movement, Select opens details, Back restores the originating item. Report what actually happened, not what should have happened. Do not claim a passing test without the evidence to back it.
